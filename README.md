# Mapeamento de Requisitos Não Funcionais

Este repositório contém um relatório técnico sobre o mapeamento de requisitos não funcionais no desenvolvimento de sistemas.

## Objetivo
Apresentar a importância dos requisitos não funcionais e explicar como eles são identificados e organizados durante o desenvolvimento de software.

## Conteúdos abordados

- Introdução aos requisitos não funcionais
- Tipos de requisitos não funcionais
- Processo de mapeamento
- Importância para a qualidade do software
